=====Media Player====

You start using the media player
by selecting Edit menu and making
a playlist.

Media player has 2 different pages:
Music and Game.

On music page you can listen music
and/or cut mp3 songs. If you want
to cut songs, you need to press
CutMod to change the program 
to cutting. Then you can press
cut to cut the are between
thumbs and save it to another
file.


On game page you start by choosing
which playlist you want to use
for the game. The program starts
to play some musich that user needs
to recognize as fast as possible. Then
you quess which song it is and get
points if you were fast and right.
